Vehicle Routing Problem

The vehicle routing problem (VRP) is a combinatorial optimization and integer programming problem which asks "What is the optimal set of routes for a fleet of vehicles to traverse in order to deliver to a given set of customers?"

For "last mile delivery", we have VRP +,

  • specific delivery windows
  • new orders constantly
  • decisions about which shoppers can do which deliveries
  • decisions about how to group deliveries together
  • many metrics that tell us how well we are doing

We write programs to understand and solve problems

We also use graphs to visualize solutions

We also write programs to keep track of our metrics
